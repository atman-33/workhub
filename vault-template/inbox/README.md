# Inbox

Landing zone for raw input. Drop new notes, clippings, and ideas here without
worrying about structure — run `/kb-ingest` to classify them into
`projects/`, `knowledge/`, or `archive/` (and to turn actionable items into
tasks).

The `_wip/` subfolder is reserved for notes that are intentionally excluded
from automatic ingestion and reorganization. Use it for scratchpads and
work-in-progress material that should stay in the inbox.

This README is a structural note: agents never ingest, move, or rewrite it.
