# README screenshots

Drop the following PNG files into this folder; the root `README.md` already
references them by these exact names. Recommended: capture the app window at
around 1280×800 (dark theme is the app default).

| File | What to capture |
|---|---|
| `tasks-kanban.png` | **Hero shot.** Tasks tab, kanban view, a few cards spread across columns (Inbox → Done), filters visible at the top. |
| `tasks-edit-dialog.png` | The Edit Task dialog of a claude-code task — markdown preview visible in Description, Results / launch buttons in the header. |
| `tasks-launch-agent.png` | A task card (or the edit dialog) with the **Launch agent** robot button visible — ideally mid "launching" animation, or with the spawned terminal next to the app. |
| `repos.png` | Repos tab with several registered repositories, branch/dirty state visible; Changes or Worktrees panel open if possible. |
| `music-timer.png` | A side-by-side or composite of the Music tab (playlist + player) and the Timer tab (countdown ring). Two separate shots stitched horizontally works fine. |
| `ink-overlay.png` | Any screen with a few ink strokes drawn on it (double-press-and-hold Alt), pen-color chip visible near the cursor. |
| `setup-vault.png` | The first-launch vault picker in the Tasks tab (or ⚙ Settings → *Tasks vault path*) showing the **Init vault** flow. |
