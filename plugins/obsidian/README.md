# obsidian

Helpers for working with Obsidian notes and knowledge management.

## Skills

| Skill | Description |
|-------|-------------|
| `defuddle` | Extract clean markdown content from web pages using Defuddle CLI, removing clutter to save tokens |
| `json-canvas` | Create and edit JSON Canvas files (.canvas) with nodes, edges, groups, and connections |
| `kb-index` | Update PARA Knowledge Base indexes — smart mode or full rebuild |
| `kb-ingest` | Ingest documents into the PARA Knowledge Base: classify, move, link, and index |
| `kb-init` | Initialize PARA Knowledge Base in your Obsidian vault (run first before other kb- skills) |
| `kb-lint` | Health check for the PARA Knowledge Base — detects orphan docs, broken links, index drift |
| `kb-query` | Query the PARA Knowledge Base to find information and synthesize answers across documents |
| `obsidian-bases` | Create and edit Obsidian Bases (.base files) with views, filters, formulas, and summaries |
| `obsidian-cli` | Interact with Obsidian vaults using the Obsidian CLI to manage notes, tasks, and properties |
| `obsidian-markdown` | Create and edit Obsidian Flavored Markdown with wikilinks, embeds, callouts, and properties |
| `zenn-blog-writing` | Zennの技術ブログ記事を執筆・編集・レビューするためのガイドライン |
| `zenn-markdown` | Zenn記法（メッセージブロック、アコーディオン、埋め込みなど）の構文リファレンス |
